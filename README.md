# MO Music Bot

بوت موسيقى ديسكورد بأوامر بدون /:

- `p` تشغيل (اسم أو رابط يوتيوب)
- `sk` تخطي
- `s` إيقاف
- `pa` إيقاف مؤقت
- `r` استئناف
- `q` عرض قائمة الانتظار
- `n` الآن يُشغّل
- `v 100` تغيير الصوت (1 - 200)

## الإعداد على Railway

1. ارفع هذا المشروع إلى GitHub.
2. اربطه مع Railway (Deploy from GitHub).
3. في صفحة Variables على Railway أضف متغير:
   - `DISCORD_TOKEN` = توكن البوت.
4. أعد تشغيل البوت.
